# Reference Assets

This folder contains reference assets for the OAD brand:

## Logo Files
- `logo-primary.svg` - Primary OAD logo (color)
- `logo-white.svg` - White version for dark backgrounds
- `logo-stacked.svg` - Stacked/vertical logo variant

## Approved Examples
Place approved design examples in `approved-examples/`:
- `banner-good.png` - Example of a compliant banner
- `banner-bad.png` - Example of a non-compliant banner
- `social-good.png` - Example of compliant social media post

## Usage
These assets are referenced by the AI agent for visual comparison during design reviews.
